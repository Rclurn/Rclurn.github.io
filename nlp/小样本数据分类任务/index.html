3、评测标准

评测采用macro F1指标，计算公式如下：

$$ F1=\frac{1}{n}\sum_{i}^{n}{F1_i} = \frac{1}{n}\sum_{i}^{n}{\frac{2\times P_i \times R_i}{P_i+R_i}} $$

Pi是第i个类别的precition，Ri是第i个类别的Recall，n是类别总数。

赛题突破点(仅供参考)

1、多分类样本不均衡

过采样

从少样本类别中反复抽取样本，直至该类别样本数与多样本类别基本持平。这样势必会在原来少样本的类别中产生大量的重复样本，如果其中有数据标签错误或者存在噪音，那么这个错误将被成倍的放大，所以过采样最大的缺点就是对少样本类别过拟合。

欠采样

抛弃大量的多样本类别样本，从多样本类别中随机抽取一部分样本来与少样本类别数量基本持平。首先，数据十分珍贵，而且这是一个小样本任务，抛弃数据的做法不可取；其次，抛弃了大部分数据，弱化了多样本类别的影响，可能造成数据偏差较大。

改造损失函数👉🏻Focal Loss

Focal Loss的核心功能是解决难样本问题，其核心参数有两个alpha与gamma。alpha是一个与类别相关的参数，如果仅使用这个参数，应该给予数量少的类别较大的权重，数量多的类别较小的权重，但是在与gamma参数共用时，根据该论文实验表明，alpha的取值应该是给予数量少的类别较小的权重，数量多的类别较大的权重，具体的解释参考苏剑林大神的解释：本来样本少的类别“难以匹敌”样本多的类别，但是经过alpha和gamma的优化后，形式逆转了，所以需要对样本多的类别降权。不过我认为alpha这个参数的引入，对于每个类别应该使用什么样的权重需要多多尝试(这里使用的是用样本数最多的样本数量依次除以每个类别的样本数)，如果在实验是发现alpha的加入对于整体的提升不是很大的话，可以尝试只引入gamma参数进行快速调参。

class MultiFocalLoss(nn.Module):
    def __init__(self, class_num, alpha=None, gamma=2, logits=False, reduce=None):
        super(MultiFocalLoss, self).__init__()
        self.alpha = alpha
        self.alpha = self.alpha.cuda()
        self.gamma = gamma
        self.class_num = class_num
        self.reduce = reduce

    def forward(self, inputs, targets):
        pt = F.softmax(inputs, dim=1)  # softmmax获取预测概率
        class_mask = F.one_hot(targets, self.class_num)  # 获取target的one hot编码
        ids = targets.view(-1, 1)
        alpha = self.alpha[ids.data.view(-1)].view(-1, 1)  # 注意，这里的alpha是给定的一个list(tensor
        # ),里面的元素分别是每一个类的权重因子
        alpha.requires_grad = True
        probs = (pt * class_mask).sum(1).view(-1, 1)  # 利用onehot作为mask，提取对应的pt
        log_p = probs.log()
        # 同样，原始ce上增加一个动态权重衰减因子
        if self.alpha is not None:
            loss = -alpha * (torch.pow((1 - probs), self.gamma)) * log_p
        else:
            loss = - (torch.pow((1 - probs), self.gamma)) * log_p
        if self.reduce == 'mean':
            loss = loss.mean()
        elif self.reduce == 'sum':
            loss = loss.sum()
        return loss

初步测试：加入alpha后的效果好于不加alpha的效果。

通用EDA：

同义词替换 从句子中随机选择非停用词，然后用这些选中词对应的同义5664词来替换它们
随机插入 随机的找出句中某个非停用词，随后使用其同义词插入句中的一个随机位置。这个操作可以重复n次
随机交换 随机的选择句子中的两个词语，然后交换这两个词语的位置。这个操作可以重复n次
随机删除 以固定概率随机删除句子中的词语
基于TF-IDF的非核心词替换
