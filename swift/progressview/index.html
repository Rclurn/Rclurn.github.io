ProgressView

2023-03-29 20:58

  • Development

Declaration

struct ProgressView<Label, CurrentValueLabel> where Label: View, CurrentValueLabel: View

概述

ProgressView通常用于展示一个未完成的任务的完成进度，它同时支持显示具体百分比和非具体类型的进度。创建一个有具体百分比的进度条，通常通过在初始化ProgressView时传递一个数值类型Binding，默认情况下，这个值初始为0.0，最大为1.0。下面是一个显示具体百分比进度条的例子，通过使用State属性来定义一个进度百分比数值变量progress作为ProgressView的初始化百分比值，使用默认的最大值1.0。因为progress初始为0.5，所以进度条一开始便显示完成了一半。进度条下面的More按钮允许使用者每次在progress原有值的基础上增加0.05。

struct LinearProgressDemoView: View {
	@State private var progress = 0.5
    
    var body: some View {
        VStack {
            ProgressView(value: progress)
            Button("More") { progress += 0.05 }
        }
    }
}

如果要创建一个没有具体进度的进度条，使用一个没有参数的初始化器即可：

var body: some View {
    ProgressView()
}

![image-20230329214339987](/Users/rnake/Library/Application Support/typora-user-images/image-20230329214339987.png)

还可以通过传递一个日期的闭区间，只要当前日期在闭区间范围内，进度视图就会随着日期的更新自动更新、填充或充满进度视图。下面是一个显示五分钟的计时器，其开始时间为进度视图的初始化时间：

struct DateRelativeProgressDemoView: View {
    let workoutDateRange = Date()...Date().addingTimeInterval(5*60)
    
    var body: some View {
        ProgressView(timerInterval: workoutDateRange) {
            Text("Workout")
        }
    }
}

设置进度条的样式

可以通过创建遵从ProgressViewStyle协议的样式来自定义进度视图的外观和交互。使用progressViewStyle(_:)修饰符可以一次性设置视图中的所有进度视图实例饿的样式，在下面的示例中，自定义样式将圆形粉红色边框添加到VStack视图的所有进度视图中：

struct BorderedProgressViews: View {
    var body: some View {
        VStack {
            ProgressView(value: 0.25) { Text("25% progress") }
            ProgressView(value: 0.75) { Text("75% progress") }
        }.progressViewStyle()
    }
}
struct PinkBorderedProgressViewStyle: ProgressViewStyle {
    func makeBody(configuration: Configuration) -> some View {
        ProgressView(configuration)
        	.padding(4)
        	.border(.pink, width: 3)
        	.cornerRadius(4)
    }
}

SwiftUI提供了两个内置的ProgressViewStyle,linear和 circular，circular只有在Self为CircularProgressViewStyle时才可以使用。下面的示例显示了一个循环进度条，该进度条从完成60%开始：

struct CircularProgressDemoView: View {
    @State private var progress = 0.6

    var body: some View {
        VStack {
            ProgressView(value: progress)
                .progressViewStyle(.circular)
        }
    }
}
